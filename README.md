# Bela Nepal Industries - Corporate Landing Page

A modern, high-performance landing page for Bela Nepal Industries, built with Next.js, Tailwind CSS, and Framer Motion. This project is a Single Page Application (SPA) designed for static export.

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ 
- npm or pnpm

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd bela-landing-page
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000/belaweb2test](http://localhost:3000/belaweb2test) to view the site.
   *(Note: The root `/` will automatically redirect to the base path `/belaweb2test`)*

4. **Build for Production:**
   ```bash
   npm run build
   ```
   The static output will be generated in the `out/` directory.

- **Responsive Design**: Mobile-first approach using Tailwind CSS.
- **Dark/Light Mode**: Built-in theme switching.
- **Bilingual**: English and Nepali language support.
- **Animations**: Smooth transitions using Framer Motion.
- **Static Export**: Optimized for hosting on any static server (GitHub Pages, Vercel, Netlify).
