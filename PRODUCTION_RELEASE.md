The following security standards have been implemented for this production release:

- **HSTS**: Strict-Transport-Security enabled (requires SSL cert).
- **Anti-Clickjacking**: `X-Frame-Options: DENY`
- **XSS Protection**: `X-Content-Type-Options: nosniff` & CSP.
- **Privacy**: `Referrer-Policy: strict-origin-when-cross-origin`

### 2. Application Security
- **Content Security Policy (CSP)**: Configured to allow only trusted sources (Self + Google Maps/Fonts).
- **Tab Nabbing**: All external links secured with `rel="noopener noreferrer"`.
- **Sanitization**: React auto-escaping prevents injection attacks.
- **Cookie Security**: No client-side cookies used for session management (Stateless).
